# AI PPT 生成器 - MVP

使用人工智能快速生成专业的演示文稿

## 功能特点

- 智能内容生成
- 专业模板设计
- 快速导出下载
- 多格式支持

## 使用方法

1. 访问 [GitHub Pages](https://xiao-woo.github.io/ai_pptx_generator_online/)
2. 输入演示主题
3. 描述内容要点
4. 点击生成按钮

## 技术栈

- HTML5
- CSS3
- JavaScript
- GitHub Pages
